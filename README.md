
RestaurantFinder - интерактивная витрина ресторанов с системой рейтинга

Цели проекта
Разработка приложения с нуля: Полный цикл создания React-приложения, от инициализации до реализации бизнес-логики.
Проектирование архитектуры: Формирование чёткой и масштабируемой структуры файлов и папок.
Работа с данными и состоянием: Закрепление навыков управления данными в React-компонентах, а также практическое применение концепций контролируемых и неконтролируемых компонентов.
Технические задачи и реализация
Компонентный подход: Создание библиотеки абстрактных UI-компонентов (поиск, карточка ресторана, логотип, аватар пользователя).
Управление данными (Lifting State Up): Логика фильтрации и изменения рейтинга вынесена на уровень родительского компонента страницы для обеспечения единства источника истины.
Сетевое взаимодействие: Реализация асинхронных запросов для получения списка ресторанов и обработки состояний загрузки (Loading states).
Интерактив: Функционал фильтрации списка в реальном времени по названию и возможность динамического изменения рейтинга заведения.


## Запуск

1. Установить зависимости:

   ```bash
   npm install
   ```

2. Запустить приложение:

   ```bash
   npm run dev
   ```
